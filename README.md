# LearnForU
Requirements - MVP1

Login (or) Signup 
1. For Users without account, Signup the User with Basic details
2. Once Signup, verify AuthN of user with email OTP.
3. Give Scope to Users to choose their Tech intrests from the displayed Options.
4. Suggest the User to follow Top Existing accounts in the App who had similar intrests to reflect the content in his feed
   

Landing Page
1. If User Chosen Accounts to follow In SignUp process, Show No. of following Accounts and Show Learning paths posted by them in feed.
2. If User didn't chose any account, then show 0 accounts followed and an Explore option to search for needed account for following.
3. Upon refresh everytime, show latest timestamp learning paths by following Accounts.
4. Enable Search option to search for accounts.

Learning Paths
1. Create - Create Learning Path with some mandatory informations as instructed for each topic.
2. Update - This option should available only for the learning paths created by user
3. Delete - This option should available only for the learning paths created by user
4. Read - All learning paths posted by accounts who followed is visible in feed.
5. Like - Like the Learning Path
6. Comment - Comment on Learning Path
7. Tag User accounts under learning paths
8. Share - Share Learning paths ( Enable sharing to multiple apps Ex: Whatsapp, Instagram)

Summarize Learning Paths for a Topic 
1. For Each Topic, There can be multiple Learning paths created by users. To Summarize all of them for each topic and create a summarized learning path. (Visible to only few users initially)
